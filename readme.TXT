Compress Your Images to Load Your Blog Images Faster
 
== Description ==
 
This plugin compresses the uploaded image file on the fly and convert it into JPG format. The original uploaded file is replaced with new compressed one so be sure before using this plugin.
 
JS Image Compressor optimizes your image and keeps the image quality same but reduces the size of your image which helps your website load faster.

 
== Installation ==

1) Actual Size of the Image
2) Once Uploaded to WP The Compressed Size of the Image
3) Before & After the Upload

= From WordPress Backend =

Follow the Steps to Install JS Image Compressor.
 
 
1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
